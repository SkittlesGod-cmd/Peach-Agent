Telegram

Your agent
in your pocket.

Connect Peach to Telegram and get live quotes, real-time alerts, portfolio P&L, and on-demand briefings from anywhere — on any device.

Open BotFather → Back to install guide
  1. 01

    Create a bot with BotFather

    Open Telegram and message @BotFather. Send the /newbot command, then follow the prompts to choose a name and username for your bot.

    # In Telegram, message @BotFather:
/newbot

# BotFather asks for a name, then a username:
My Peach Bot
myPeachBot_bot

# BotFather replies with your token:
# 1234567890:ABCdef-ghijklmnopqrstuvwxyz1234567

    The bot token is a secret. Don't share it or commit it to git. Peach reads it from your config file, which is gitignored.

  2. 02

    Add the token to your config

    Open ~/.local/share/peach-agent/peach_config.json and add the token BotFather gave you:

    {
  "telegram_bot_token": "1234567890:ABCdef-ghijklmnopqrstuvwxyz1234567"
}

    You can edit the file with any text editor, or run:

    nano ~/.local/share/peach-agent/peach_config.json
  3. 03

    Restart Peach

    Stop and restart the daemon so it picks up the new config and starts the Telegram bot thread:

    peach stop && peach start
  4. 04

    Activate your bot

    Open Telegram and search for the username you set in BotFather. Send the /start command. Peach will reply with a list of commands and automatically save your chat ID so future notifications reach you.

    # In your Telegram chat with the bot:
/start

    The first /start message saves your chat ID automatically. After that, Peach will push morning briefing summaries and price alert notifications directly to you — no extra setup needed.

  5. 05

    Add notifications to a second device Optional

    Run /mychatid to see your chat ID. Add it to the config for proactive notifications (price alerts, briefing summaries) even when you haven't messaged the bot recently:

    # Send in Telegram:
/mychatid

# Then add to peach_config.json:
{
  "telegram_bot_token": "YOUR_TOKEN",
  "telegram_chat_id":   "123456789"
}

What you can ask

The bot has access to nine tools. Type any natural-language question and it will use them automatically. Or use the slash commands directly:

/briefing Run a full morning briefing right now using live market data, technicals, and portfolio context.
/pdf Generate and send a full PDF morning brief with macro table, portfolio, and embedded charts.
/chart AAPL [3mo] Price candlestick with volume. Pass multiple tickers for a normalized return comparison: /chart AAPL MSFT NVDA 3mo.
/technicals AAPL RSI(14), SMA20, SMA50, 52-week high/low, and distance from each level.
/correlation Send a heatmap of 30-day return correlations for your portfolio holdings.
/history Portfolio value over time — market value vs cost basis with daily snapshots going back 90 days.
/quote AAPL Current price, day change, and volume.
/portfolio All open positions with current price, unrealised P&L, and thesis notes.
/add AAPL 10 150.00 Add 10 shares at $150 cost basis. Optionally append a thesis. Re-adding averages the cost basis automatically.
/remove AAPL Remove a position from the portfolio ledger.
/alert AAPL above 200 Price alert — Peach checks every 5 minutes during market hours and fires instantly on trigger.
/alerts List all active price alerts.
/trade LONG NVDA 900 AI play Log a paper trade. Specify LONG or SHORT, ticker, entry price, optional shares and thesis.
/trades View your paper trade history with entry/exit prices and P&L for closed trades.
/closetr 3 950 Close paper trade #3 at $950. Calculates and displays the P&L.
/mychatid Show your Telegram chat ID. Needed for proactive notifications if auto-detection fails.
Free text Ask anything: "what's happening with NVDA?", "is my portfolio up?", "any earnings this week?". The agent picks the right tools and answers.

What the agent can do

The Telegram bot uses the same agentic loop as the morning briefing. For each message, it can call up to eight tools before formulating a reply:

  • get_quote — current price, day change, and volume for any ticker
  • get_portfolio — all positions with live unrealised P&L
  • get_pre_market — pre-market price and percentage change (before 9:30 AM ET)
  • get_news — recent headlines for a specific ticker from Yahoo Finance
  • get_technicals — RSI(14), SMA20/50, 52-week high/low, trend signals
  • get_earnings_calendar — watchlist tickers reporting earnings in the next 7 days
  • get_correlation — 30-day return correlation matrix for portfolio holdings
  • search_web — real-time DuckDuckGo search for breaking news or analyst notes
  • add_alert — set a price threshold alert from conversation context

The agent reasons about which tools to call. Ask "is NVDA overbought?" and it calls get_technicals. Ask "any earnings risk this week?" and it calls get_earnings_calendar then search_web for analyst expectations.

Alerts

Set it and
forget it.

# Set an alert from Telegram:
/alert NVDA above 1100

# Or just type naturally:
"alert me if SPY drops below 490"

# Peach checks every 5 minutes (market hours) and pushes:
# "Alert fired: NVDA is above $1100 (now $1104.30)"

Alerts are stored in SQLite and survive daemon restarts. Once fired, they are marked done and won't repeat.
