Connect Peach to Telegram and get live quotes, real-time alerts, portfolio P&L, and on-demand briefings from anywhere — on any device.

Open BotFather → Back to install guide
  1. 01

    Create a bot with BotFather

    Open Telegram and message @BotFather. Send the /newbot command, then follow the prompts to choose a name and username for your bot.

    # In Telegram, message @BotFather:
/newbot

# BotFather asks for a name, then a username:
My Peach Bot
myPeachBot_bot

# BotFather replies with your token:
# 1234567890:ABCdef-ghijklmnopqrstuvwxyz1234567

    The bot token is a secret. Don't share it or commit it to git. Peach reads it from your config file, which is gitignored.

  2. 02

    Add the token to your config

    Open ~/.local/share/peach-agent/peach_config.json and add the token BotFather gave you:

    {
  "telegram_bot_token": "1234567890:ABCdef-ghijklmnopqrstuvwxyz1234567"
}

    You can edit the file with any text editor, or run:

    nano ~/.local/share/peach-agent/peach_config.json
  3. 03

    Restart Peach

    Stop and restart the daemon so it picks up the new config and starts the Telegram bot thread:

    peach stop && peach start
  4. 04

    Activate your bot

    Open Telegram and search for the username you set in BotFather. Send the /start command. Peach will reply with a list of commands and automatically save your chat ID so future notifications reach you.

    # In your Telegram chat with the bot:
/start

    The first /start message saves your chat ID automatically. After that, Peach will push morning briefing summaries and price alert notifications directly to you — no extra setup needed.

  5. 05

    Add notifications to a second device Optional

    Run /mychatid to see your chat ID. Add it to the config for proactive notifications (price alerts, briefing summaries) even when you haven't messaged the bot recently:

    # Send in Telegram:
/mychatid

# Then add to peach_config.json:
{
  "telegram_bot_token": "YOUR_TOKEN",
  "telegram_chat_id":   "123456789"
}

What you can ask

The bot has access to five live tools. Type any natural-language question and it will use them automatically. Or use the slash commands directly:

/briefing Run a full morning briefing right now using live market data and your portfolio context.
/quote AAPL Get the current price, day change, and volume for any ticker.
/portfolio See all open positions with current price, unrealised P&L, and your thesis notes.
/add AAPL 10 150.00 Add 10 shares of AAPL at a $150 cost basis. Add a thesis at the end: /add AAPL 10 150 AI supercycle play. If you already hold AAPL, the cost basis is averaged automatically.
/remove AAPL Remove a position from your portfolio.
/alert AAPL above 200 Set a price alert. Peach checks every 5 minutes during market hours and fires a notification the moment the price crosses.
/alerts List all active price alerts.
/account Show your Alpaca account summary: total equity, cash, buying power, and today's P&L. Requires Alpaca credentials in config.
/broker Show all open positions in your Alpaca account with real cost basis, current price, total unrealised P&L, and intraday P&L. Requires Alpaca credentials in config.
/mychatid Show your Telegram chat ID. Useful for configuring proactive notifications.
Free text Ask anything in plain language — "what's happening with NVDA today?", "is my portfolio up or down?", "set an alert if SPY drops below 500". The agent calls the right tools and answers.

What the agent can do

The Telegram bot uses the same agentic loop as the morning briefing. For each message, it can call up to eight tools before formulating a reply:

  • get_quote — current price, day change, volume for any ticker
  • get_portfolio — all positions with live P&L calculations
  • get_pre_market — pre-market price and percentage change (before 9:30 AM ET)
  • get_news — recent headlines for a specific ticker from Yahoo Finance
  • add_alert — set a threshold alert from conversation context

The agent reasons about what tools to use based on your question. If you ask "how is my tech exposure looking?", it calls get_portfolio and get_quote for each holding before answering.

Alerts

Set it and
forget it.

# Set an alert from Telegram:
/alert NVDA above 1100

# Or just type naturally:
"alert me if SPY drops below 490"

# Peach checks every 5 minutes (market hours) and pushes:
# "Alert fired: NVDA is above $1100 (now $1104.30)"

Alerts are stored in SQLite and survive daemon restarts. Once fired, they are marked done and won't repeat.
