Built for quiet, repeatable market prep.

Peach is a modular Python agent with a CLI controller, daemon scheduler, data aggregation, analysis, and email notification pipeline.

Detached CLI control

Launch Peach from a terminal, keep it running in the background, and stop it cleanly through a PID-backed controller.

Market data pipeline

Fetch previous-session closes, volumes, percentage changes, and recent headlines for your configured watchlist.

LLM-ready briefing

Send a structured prompt to OpenAI or Ollama and receive a Markdown report with macro tape, standouts, sentiment, and a lookout list.

The loop

A small set of composable modules keeps Peach easy to audit, run, and move to a real always-on host.

One line to install.

Install Peach on any machine with Python 3.9+ and git. The script clones the repo, sets up a virtual environment, and drops a peach command on your PATH.

curl -fsSL https://raw.githubusercontent.com/SkittlesGod-cmd/Peach-Agent/main/install.sh | bash

Then export your credentials and start the daemon:

export SMTP_USERNAME="you@example.com"
export SMTP_PASSWORD="your-app-password"
export PEACH_EMAIL_FROM="you@example.com"
export PEACH_EMAIL_TO="you@example.com"
export PEACH_LLM_PROVIDER="openai"
export OPENAI_API_KEY="sk-..."

peach start
peach status
