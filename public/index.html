What it does

Quiet, repeatable market prep.

Peach is a modular Python agent — a scheduler, data fetcher, LLM analysis engine, Telegram bot, and delivery pipeline in a single package. Runs in the background. Needs no babysitting.

  • Morning briefing

    At 09:00 ET every weekday, Peach fetches market data, builds an analyst prompt, and calls the pre-configured LLM. The result is a structured Markdown report delivered to email or a local file.

  • Telegram bot

    Ask for live quotes, portfolio P&L, on-demand briefings, and price alerts — all from Telegram. The agent uses tool calls to fetch live data before answering.

  • Portfolio tracking

    Add positions with a cost basis and thesis. Peach computes real-time P&L against current prices whenever you ask.

  • Price alerts

    Set threshold alerts — Peach checks every 5 minutes during market hours and pushes a Telegram notification the moment the price crosses.

  • Deterministic fallback

    If the LLM provider is unreachable, Peach generates a data-first briefing from raw metrics. A provider outage will never crash the daemon or silence the morning report.

The agent loop

Reason. Call. Re-reason.

Peach doesn't just fill in a template. It runs a tool-use loop — calling tools for live data, reading the results, and continuing to reason until the answer is complete.

  1. 01

    Schedule

    APScheduler fires the pipeline at 09:00 ET on weekdays. The schedule, timezone, and tickers are fully configurable.

  2. 02

    Fetch

    Previous-session closes, volumes, pre-market moves, and headlines are collected. Individual failures are logged without stopping the run.

  3. 03

    Reason

    The market snapshot and portfolio context are given to the agent. The LLM decides what tools to call — live quotes, pre-market data, news — before writing the briefing.

  4. 04

    Tool calls

    The agent calls up to 8 tools per briefing. Each result is fed back into the reasoning loop before the next step.

  5. 05

    Deliver

    The finished briefing is formatted as a Markdown email and delivered via SMTP — or saved to a local file if no email is configured.

  6. 06

    Notify

    A summary is pushed to Telegram, and price alerts are checked every 5 minutes during market hours for the rest of the day.

Install

Two commands.
Nothing else.

# 1. Install
curl -fsSL https://raw.githubusercontent.com/SkittlesGod-cmd/Peach-Agent/main/install.sh | bash

# 2. Start
peach start

No API keys to set. No config files to edit. Briefings land in ~/briefing.md until you add email or Telegram.
