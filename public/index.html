What it does

Automated research
outreach.

Peach is a full agentic pipeline — web search, lab page scraping, LLM-powered email writing, SQLite tracking, and a live dashboard. It runs in the background and handles the entire outreach workflow so you can focus on what matters.

  • 01

    Real web search

    Peach doesn't guess from training data. It searches DuckDuckGo for real faculty directory pages and lab profiles, finding professors active in neuroscience and neurosurgery right now.

  • 02

    Lab page scraping

    Each faculty page is fetched and parsed. Peach extracts email addresses, recent paper titles, and research focus — the raw material for a genuinely personalized email.

  • 03

    Personalized outreach

    Every email references a specific paper by name, connects your background to their research, and stays under 200 words. No generic templates — each one reads like you wrote it.

  • 04

    Automated follow-up

    Professors who haven't replied after 14 days get a polite follow-up automatically. Peach tracks the cadence so you never have to remember who's waiting.

  • 05

    Research dashboard

    A live web dashboard shows every professor found, their status (found / emailed / followed-up / replied), and filters by university or subfield. Synced from your local daemon in real time.

  • 06

    Nationwide coverage

    45+ programs including Johns Hopkins, Stanford, UCSF, Harvard, Yale, Columbia, Mayo, and more — across neurosurgery, brain tumor research, computational neuro, and neuroimaging.

The agent loop

Search. Scrape.
Email. Track.

Peach doesn't fill in a template. It runs a full tool-use loop — searching the real web, reading actual lab pages, writing personalized emails, and syncing to your dashboard.

  1. 01

    Web search

    Peach queries DuckDuckGo for faculty directories and lab pages at each target university, using subfield-specific search terms to find active researchers.

  2. 02

    Scrape

    Each promising URL is fetched. The agent extracts the professor's email, recent publications, lab focus, and department — the foundation for a real email.

  3. 03

    Discover

    Real professors with confirmed data are saved to a local SQLite database. Duplicates are detected automatically — each professor is tracked once.

  4. 04

    Email

    If a real email was found on the page, Peach drafts and sends a cold email referencing a specific paper. The email is under 200 words and written for your exact background.

  5. 05

    Sync

    After each hunt, the full professor database is pushed to the Vercel dashboard so your outreach tracker stays current from any device.

  6. 06

    Follow-up

    Professors emailed 14+ days ago with no reply are surfaced automatically. A polite follow-up is sent and the status is updated in the database.

Install

Three commands.
Then it hunts.

# 1. Install
curl -fsSL https://raw.githubusercontent.com/SkittlesGod-cmd/Peach-Agent/main/install.sh | bash

# 2. Start the daemon
peach start

# 3. Run your first research hunt (3 universities)
peach research hunt --batch 3

No API keys. No accounts. Your student profile in peach_config.json is all Peach needs to personalize every email.
