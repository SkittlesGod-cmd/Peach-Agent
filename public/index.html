What it does

Quiet, repeatable market prep.

Peach is a modular Python agent — a CLI controller, daemon scheduler, data aggregation layer, LLM analysis engine, and email delivery pipeline in a single package.

  • Detached CLI control

    Launch from a terminal, keep running in the background, stop cleanly. A PID file tracks the process across sessions.

  • Market data pipeline

    Previous-session closes, volumes, and percentage changes for your configured watchlist via Yahoo Finance. Headlines from Alpha Vantage, NewsAPI, or RSS fallback.

  • LLM-backed briefing

    A structured prompt is sent to OpenAI or a local Ollama model. The result is a Markdown report with macro tape, standouts, sentiment, and a lookout list.

  • Deterministic fallback

    If the LLM provider is unreachable, Peach generates a data-first briefing from raw metrics so the daemon never crashes silently.

The loop

Five composable steps.

  1. 01

    Schedule

    APScheduler fires the pipeline at 09:00 ET on weekdays. The schedule, timezone, and tickers are configurable per deployment.

  2. 02

    Fetch

    Market metrics and headlines are collected with guarded network calls. Individual fetcher failures are logged without stopping the run.

  3. 03

    Analyze

    The payload is shaped into a focused analyst prompt and sent to your chosen LLM. Instructions are scoped to the data provided — no hallucinated prices.

  4. 04

    Notify

    The Markdown report is formatted as a multipart email and delivered via SMTP. HTML and plaintext parts are both included.

  5. 05

    Log

    Rotating file logs capture every run. Failures degrade the briefing gracefully — they never terminate the daemon process.

Install

One command.

curl -fsSL https://raw.githubusercontent.com/SkittlesGod-cmd/Peach-Agent/main/install.sh | bash

Requires Python 3.9+ and git. Installs to ~/.local/share/peach-agent and drops a peach command on your PATH.

# Set credentials, then:
peach start
peach status
peach stop
