Getting started

Does Peach need an API key?

No. The LLM runs through a pre-configured Vercel proxy backed by free OpenRouter models. You never touch an API key. If you want to use your own key, set llm_provider to "openrouter" in the config and add OPENROUTER_API_KEY.

What Python version do I need?

Python 3.9 or later. Check with python3 --version. Python 3.11+ is recommended for best performance.

Does it work on Windows?

Peach is tested on macOS and Linux. It may work on Windows with WSL2 (Windows Subsystem for Linux), but is not officially supported.

How long does installation take?

About 30 seconds. The script clones the repository, creates a virtualenv, and installs ~20 dependencies. It prints a progress indicator the whole time.

Running & scheduling

Does Peach need to run 24/7?

The scheduler runs inside a background daemon (peach start). If your machine is off or sleeping at 09:00 ET, that day's briefing is skipped — there is no catch-up. For reliable delivery, run Peach on a VPS or always-on server. See the Deploy guide.

Can I change the briefing time?

Yes. Set schedule_hour and schedule_minute in peach_config.json. Values are in 24-hour format and respect the timezone setting (default America/New_York).

What happens if the LLM proxy is down?

Peach falls back to a deterministic data-first briefing — prices, technicals, and headlines formatted without an LLM. The briefing is clearly marked as a fallback so you know the difference.

Can I run a briefing immediately without waiting for 09:00?

Yes. Either run peach run for a one-off briefing, or start the daemon with PEACH_RUN_ON_START=true peach start.

You need a Discord server where you have permission to add a bot. Create one for free at discord.com. Then go to discord.com/developers to create a bot application, get your token, and invite it.

Data & privacy

Is my portfolio data sent anywhere?

No. Portfolio positions, paper trades, and price alerts are stored entirely in a local SQLite database at ~/.local/share/peach-agent/. Nothing leaves your machine except outbound API calls to Yahoo Finance (market data) and the Peach LLM proxy (briefing generation).

What market data source does Peach use?

Yahoo Finance via the yfinance Python library. No paid subscriptions or data API keys are required. Data is fetched over the public Yahoo Finance endpoints.

Is Peach open source?

Yes. MIT license. The full source is on GitHub at github.com/SkittlesGod-cmd/Peach-Agent. You can read every line, fork it, and extend it.
