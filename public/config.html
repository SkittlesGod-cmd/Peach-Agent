Configuration

Providers, tickers, schedule, and mail.

Peach reads environment variables first, then falls back to peach_config.json in the configured Peach home directory.

Required Email Variables

export SMTP_USERNAME="you@example.com"
export SMTP_PASSWORD="your-smtp-app-password"
export PEACH_EMAIL_FROM="you@example.com"
export PEACH_EMAIL_TO="you@example.com"

LLM Providers

ProviderVariables
OpenAIPEACH_LLM_PROVIDER=openai, OPENAI_API_KEY, PEACH_OPENAI_MODEL
OllamaPEACH_LLM_PROVIDER=ollama, OLLAMA_URL, PEACH_OLLAMA_MODEL

Market News

Alpha Vantage and NewsAPI keys are optional. If neither is configured, Peach attempts Yahoo Finance RSS as a fallback headline source.
