Deployment

Host the site on Vercel. Run the daemon somewhere always-on.

This static website deploys cleanly on Vercel. The market briefing scheduler is a persistent Python process and needs a host that stays awake.

Good Hosts for the Daemon

Cloud VM

A small Ubuntu VPS is the simplest reliable path for 24/7 execution.

Home server

A machine that stays powered and networked can run Peach locally.

Container host

Run Peach in Docker with environment-backed secrets and restart policies.

launchd/systemd

Use your OS supervisor to restart Peach after host reboots.

One-Line Install

Works on Linux and macOS. Requires Python 3.9+ and git.

curl -fsSL https://raw.githubusercontent.com/SkittlesGod-cmd/Peach-Agent/main/install.sh | bash
