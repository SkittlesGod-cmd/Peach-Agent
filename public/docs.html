Documentation

Install Peach and make it useful.

The project is a Python package with a console command. Install it, add credentials, then let the daemon handle the weekday briefing run.

Quick Install

Run this on any machine with Python 3.9+ and git. The script clones the repo, creates a virtual environment, installs the peach command, and copies the example config.

curl -fsSL https://raw.githubusercontent.com/SkittlesGod-cmd/Peach-Agent/main/install.sh | bash

Configure Credentials

Peach reads environment variables first. Set these before starting:

# Email delivery (required)
export SMTP_USERNAME="you@example.com"
export SMTP_PASSWORD="your-app-password"
export PEACH_EMAIL_FROM="you@example.com"
export PEACH_EMAIL_TO="you@example.com"

# LLM provider — OpenAI:
export PEACH_LLM_PROVIDER="openai"
export OPENAI_API_KEY="sk-..."

# — or Ollama (local, no API key needed):
export PEACH_LLM_PROVIDER="ollama"
export OLLAMA_URL="http://127.0.0.1:11434"

Credentials can also be set in peach_config.json in the install directory (~/.local/share/peach-agent/). Environment variables always take precedence.

Run

peach start     # start the background daemon
peach status    # check if it's running
peach stop      # stop and clean up the PID file

To trigger a briefing immediately (without waiting for 9:00 AM):

PEACH_RUN_ON_START=true peach start

Manual Install (from source)

git clone https://github.com/SkittlesGod-cmd/Peach-Agent.git
cd Peach-Agent
python3 -m venv .venv-peach
source .venv-peach/bin/activate
pip install -r requirements-peach.txt
pip install .
cp peach_config.example.json peach_config.json

What Peach Sends

  • Macro tape overview from the supplied market payload.
  • Unusual price and volume standouts from the configured ticker list.
  • Headline-backed sentiment notes.
  • A 2-3 ticker lookout list with technical justifications.
  • Risk notes and uncertainty when APIs or LLM providers are unavailable.

Peach has a deterministic fallback briefing if the LLM provider is down, so a provider outage does not crash the daemon.
