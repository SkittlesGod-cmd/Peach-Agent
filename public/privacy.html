Privacy

Simple, local,
transparent.

Peach is open source. You can read every line of code. Here's a plain-English summary of what touches what data.

This website

The peach-agent.vercel.app website uses no analytics, no cookies, no tracking pixels, and no third-party scripts beyond Google Fonts (for typography). No personal data is collected when you visit.

The Peach daemon (local)

Peach runs entirely on your machine. Config, portfolio positions, paper trades, price alerts, and the briefing history are stored in ~/.local/share/peach-agent/. No data is synced to any remote server.

Market data

To generate briefings, Peach calls Yahoo Finance (via the open-source yfinance library) for prices, history, and headlines. These are public, unauthenticated requests. No personally identifiable information is included.

LLM proxy

Briefing generation routes through a Vercel serverless function at peach-agent.vercel.app/api/analyze. The function forwards your market data snapshot to OpenRouter and returns the model's response. The OpenRouter API key is stored only in Vercel's environment and is never visible to you or included in any response. OpenRouter's own privacy policy applies to model inference.

Discord

If you configure a Discord bot, your discord_token and discord_channel_id are stored locally in peach_config.json. Peach uses these to send messages to your Discord channel. Discord's own privacy policy applies.

Your config file

peach_config.json is stored at ~/.local/share/peach-agent/peach_config.json. By default it is world-readable (mode 644). Peach warns you at startup if this is the case. Run chmod 600 ~/.local/share/peach-agent/peach_config.json to restrict access. Never commit this file to a public repository.
